# chirps-api

REST API for chatting app chirps

## Data Dictionary

### User

| Field      | Data Type    |
| ---------- | ------------ |
| id         | serial       |
| first_name | varchar(100) |
| last_name  | varchar(100) |
| username   | varchar(100) |
| email      | varchar(255) |
| password   | varchar(255) |
| status     | varchar(255) |
| created_at | timestamp    |
| updated_at | timestamp    |
| deleted_at | timestamp    |
